# Intern Manager
Intern Tracker is a web application designed to streamline the management of internships. It provides a platform for administrators to manage tasks and interns, while allowing interns to track their progress and complete assigned tasks.
## Key Features
### For Administrators
- **Intern Management**: Create, update, and manage intern profiles.
- **Task Assignment**: Define specific tasks and assign them to individual interns.
- **Progress Oversight**: Monitor completion status and system activity from a central dashboard.
### For Interns
- **Personal Dashboard**: View all currently assigned tasks.
- **Task Interaction**: Easily mark tasks as completed once finished.
- **Profile Tracking**: Keep track of personal progress within the program.
## Tech Stack
- **Framework**: [Laravel 12.x](https://laravel.com)
- **Frontend**: [Livewire 3.x](https://livewire.laravel.com) & [Volt](https://livewire.laravel.com/docs/volt)
- **Styling**: [Tailwind CSS](https://tailwindcss.com)
- **Asset Management**: [Vite](https://vitejs.dev)
- **Authentication**: [Laravel Breeze](https://laravel.com/docs/breeze)
